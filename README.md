## Overview

This dashboard simulates a real-world issue tracking system similar to tools like Jira or Linear. Users can view, manage, and organize project issues through a clean, responsive interface.

The primary goal of this project is to showcase:
- Scalable React application architecture
- Predictable global state management
- API-based data flows
- Professional UI and UX patterns

---

## Key Features (Planned)

- View and manage project issues in a dashboard layout  
- Create, edit, and delete issues with form validation  
- Filter issues by status and priority  
- Sort and paginate large issue lists  
- Global state management using Redux Toolkit  
- Async data fetching with loading and error states  
- Responsive, mobile-friendly layout
